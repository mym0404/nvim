# Xcode

Download [xcode-config.tar.gz](/xcode-config.tar.gz)

```shell
curl -O https://devenv.mjstudio.net/xcode-config.tar.gz
```

unzip the archive and move to the xcode config directory

```shell
export XPATH=~/Library/Developer/Xcode/UserData
rm -rf $XPATH/CodeSnippets $XPATH/FontAndColorThemes $XPATH/KeyBindings
tar -xzf xcode-config.tar.gz && cp -r xcode-config/* $XPATH/ && rm -rf xcode-config
```

## For upload

To upload your changes, you can use the following command:

```shell
cd ~/Downloads
mkdir -p xcode-config
export XPATH=~/Library/Developer/Xcode/UserData
cp -r $XPATH/CodeSnippets $XPATH/FontAndColorThemes $XPATH/KeyBindings xcode-config/
tar -czf xcode-config.tar.gz xcode-config
mv xcode-config.tar.gz ~/.config/nvim/devenv/public
rm -rf xcode-config
```
